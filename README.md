# Full Stack Open – Exercise Solutions

This repository contains my solutions to the exercises from the University of Helsinki’s Full Stack Open course.

## About the course
Full Stack Open covers modern web development with topics like:
- JavaScript
- TypeScript
- React
- Node.js / Express
- REST APIs
- Testing
- Databases
- Authentication
- CI/CD and deployment (later parts)

Course page: https://fullstackopen.com/en/

## Repo structure
Exercises are organized by course part (e.g., `part0`, `part1`, `part2`, etc.).  
Each folder contains my solutions for that part’s exercises.

## Notes
- This is for learning/practice.
- Solutions may be updated as I progress through the course.
